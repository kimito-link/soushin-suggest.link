2. `index.html`は巨大ファイル（base64画像内蔵）。**Readツールで全文読みしない**。以下の行番号をGrepまたはsedで確認してから、Editツールで一意な文字列を指定して局所編集する:

   - L2347-2353: 凡例side項（`<div class="mouse-legend-item side">`から対応する`</div>`まで）

4. 編集後、**必ずブラウザプレビューで目視確認**:
   - サイドボタンの紫い四角が実際にマウス本体の左側面から突出して見えるか
