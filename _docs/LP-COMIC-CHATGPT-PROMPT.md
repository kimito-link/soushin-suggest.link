## 6コマそれぞれの内容（共通スタイル指定の後に続けて渡す）

**1コマ目**（オレンジのラベル「1」）
```
本体のりんくがパソコンの前に座っていて、AIチャットの回答画面を見ながら困っている様子。吹き出しで
「これ、どう返信しよう…」
画面の右下隅に、サポート役の顔だけキャラ（りんくの half-eyes系表情）を小さく添えて、「うーん、どうする？」
とつぶやかせる
キャプション: 「気になる画面を見つける」
```

**2コマ目**（青のラベル「2」）
```
本体のりんくがマウスを持ち、サイドボタンを押している。画面には「カシャッ」という効果と、画面全体の
スクリーンショットが撮られる様子。吹き出しで「サイドボタン、ポチッと」
画面の隅に、サポート役の顔だけキャラ（こん太の smile系表情）を小さく添えて、「サイドボタンが無くても、
Ctrl+クリックで同じことができるよ」とコメントさせる
キャプション: 「サイドボタンでスクショ（無ければCtrl+クリックでもOK）」
```
※このコマは「サイドボタンが無いマウスでも同じことができる」という事実をLP側の本文でも必ず補足すること。
サイドボタンは複数ある選び方の一つであり、必須の操作ではない（基本はマウス操作＋Ctrlキーだけで完結する）。

**3コマ目**（緑のラベル「3」）
```

示す小さなチェックマークやキラキラエフェクト。吹き出しで「なぞるだけで、コピー完了」
画面の隅に、サポート役の顔だけキャラ（たぬ姉の smile系表情）を小さく添える
```

**5コマ目**（黄色のラベル「5」）

## 生成時の注意

- **全身参照画像(`link-full-body-ref.png`)と顔素材26枚を、6コマすべての依頼で毎回添付する**。
  1回目だけ添付して以後は省略する、というやり方だと会話が進むにつれて絵柄がずれやすい
- 6枚とも同じチャット・同じセッションで連続して生成すると、絵柄の統一を保ちやすい
- 背景色やラベルの色は多少ズレても問題ない（LP側で並べる際は個別カードとして扱うので、1枚ごとの
  完全な統一より、各コマ単体で見やすいことを優先してよい）
- 「本体のりんく」の服装・髪型が全身参照画像と一致しているか、毎回の生成結果を隅々まで見比べて
  確認すること（色・柄・靴下の左右差・髪飾りの数まで）。文章での言い換えに頼って確認を省略しない
